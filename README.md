# Đây là một trang web dùng để đăng post

## Các chức năng
- Thêm, xóa, sửa post
- Duyệt đăng post
- Thêm và sửa User
- Phân quyền User
- Đăng nhập đăng kí
- Quên mật khẩu

## Hướng dẫn cài đặt

### Server
- Dùng lệnh npm install hoặc yarn để cài đặt cái package
- Cài đặt những biến môi trường cần thiết, hãy nhìn vào phần .env.example
- MONGODB_URL: dùng để kết nối với MongoDB
- SECRET_SESSION: chữ ký bí mật dùng để tạo session và cookie
- Tạo một file .env ở trong thư mục server và follow những bước trên
- Sau khi tạo xong thì hãy chạy lệnh npm run dev

### Client và Admin
- Dùng lệnh npm install hoặc yarn để cài đặt cái package
- Đi đến thư mục và dùng lệnh npm run dev để khởi động
